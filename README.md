# Nio — The AI Operator that Remembers. 🛰️

Nio is a professional-grade AI runtime designed for autonomous, memory-centric software orchestration. Unlike traditional chat interfaces, Nio operates through a durable cognitive loop: **Analyze, Recall, Plan, Execute, and Remember.**

## 🚀 Key Features

*   **Memory Fabric**: Durable long-term memory that allows Nio to remember architectural patterns and past decisions across sessions.
*   **Cognitive Loop**: A structured execution cycle that ensures every code change is intentional, tested, and context-aware.
*   **Platform Support**: Native high-performance binaries for Windows, macOS (Universal), and Linux.
*   **Security First**: Built with Rust for memory safety. Operates in a local-first sandbox with strict policy controls.
*   **Extension Ready**: Seamless integration with Visual Studio Code.

## 📦 Installation

Download the latest binary for your platform

### Windows
1. Download `nio.exe`.
2. If Windows SmartScreen appears, click **"More Info"** and then **"Run anyway"**.
3. Move `nio.exe` to a folder in your PATH to use it from any terminal.

### macOS / Linux
1. Download the zip/tarball and extract the binary.
2. Grant execution permissions: `chmod +x nio`
3. Run with `./nio`.
